# PIC18F4550 Temperature Monitoring System

## Overview

This project implements a temperature monitoring system using the PIC18F4550 microcontroller and LM35 temperature sensor. The analog output from the LM35 is converted into digital data using the built-in ADC module of the PIC18F4550 and displayed on a 16x2 LCD. The project was designed and tested in Proteus simulation.

## Features

* Real-time temperature monitoring
* ADC-based sensor interfacing
* 16x2 LCD display output
* Accurate temperature measurement using LM35
* Proteus simulation-based implementation

## Components Used

* PIC18F4550 Microcontroller
* LM35 Temperature Sensor
* 16x2 LCD Display
* Proteus Design Suite
* Embedded C (MPLAB C18)

## Working Principle

The LM35 temperature sensor produces an analog voltage proportional to the surrounding temperature. The PIC18F4550 reads this analog signal through its internal ADC module, converts it into a digital value, calculates the corresponding temperature, and displays the result on the LCD in real time.
